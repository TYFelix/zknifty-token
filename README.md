    <p><b>zknifty</b>: Zero-knowledge transactions of non-fungible tokens on Ethereum </p>
</div>

zknifty is an experiment in using an implementation of zero-knowledge merkle tree proofs to facilitate bulk private transactions of non-fungible tokens on Ethereum. It was built as a hack at [ETHBerlin](https://ethberlin.com).
